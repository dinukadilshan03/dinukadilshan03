### 2) Flight Operations Analytics Platform
**End-to-end data warehousing and business intelligence project combining ETL, dimensional modeling, OLAP, and reporting**

**Stack:** SQL Server, SSIS, SSAS, Power BI, Excel OLAP

#### Sub-parts:
- **Data Warehouse Design**
  - Designed a **galaxy schema / fact constellation** for airline analytics
  - Modeled fact and dimension tables across ticketing and maintenance domains
  - Implemented **SCD Type 2** for passenger history tracking

- **ETL Pipeline with SSIS**
  - Built ETL workflows to ingest data from **CSV**, **text files**, and **relational sources**
  - Loaded data into staging, transformed it, and populated the warehouse
  - Extended the fact model with an **accumulating snapshot** pattern

- **SSAS Cube**
  - Created a multidimensional cube for analytical querying
  - Configured dimensions, hierarchies, measure groups, and cube relationships
  - Enabled **roll-up**, **drill-down**, **slice**, **dice**, and **pivot** style analysis

- **BI & Reporting**
  - Built Power BI reports for **revenue trends**, **route analysis**, and **aircraft maintenance**
  - Used Excel for OLAP-style cube exploration and reporting workflows

🔗 [ETL / DW Repository](https://github.com/dinukadilshan03/FlightOperationsDW-Project)  
🔗 [SSAS Repository](https://github.com/dinukadilshan03/FlightOps_SSAS_Project)

### 3) lr-rag
**Local Retrieval-Augmented Generation system for grounded question answering over PDF documents**

**Stack:** Python, Ollama, Qdrant, LlamaIndex, Streamlit

- Built a **self-hosted RAG pipeline** for PDF ingestion, chunking, embedding, retrieval, and grounded answer generation
- Used **Qdrant** as the vector database and **Ollama** for local model inference
- Designed the system to prioritize **privacy**, **local execution**, and **citation-grounded responses**
- Implemented both ingestion and retrieval pipelines in a modular way

🔗 [Repository](https://github.com/dinukadilshan03/lr-rag)
